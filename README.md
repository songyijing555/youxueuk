# 英国游学网 - UK Study Tour Website

## 项目描述

这是一个专业的英国游学网站首页，展示三个主要服务项目：
- 短期游学
- 私校插班
- 暑期学校

## 功能特性

✨ **核心功能**
- 响应式设计 - 适配各种设备屏幕
- 导航栏 - 快速导航到不同部分
- 英雄部分 - 突出的背景和主题
- 服务展示卡 - 详细展示三个主要项目
- 咨询按钮 - 多处CTA (行动号召)
- 咨询表格 - 模态框咨询表单
- 联系信息 - 页脚包含联系方式

## 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 响应式样式设计
- **JavaScript** - 交互功能

## 项目结构

```
uk-study-tour/
├── index.html          # 主页面
├── css/
│   └── styles.css      # 样式表
├── js/
│   └── script.js       # JavaScript脚本
├── images/             # 图片资源目录(占位符)

```

## 快速开始

### 方式一: 直接打开
1. 在浏览器中打开 `index.html` 文件

### 方式二: 使用本地服务器 (推荐)

**使用Python (Python 3):**
```bash
python -m http.server 8000
```

**使用Node.js (需要安装http-server):**
```bash
npx http-server
```

然后在浏览器访问: `http://localhost:8000`

## 功能说明

### 导航栏
- 包含网站LOGO和主导航菜单
- 顶部咨询按钮，便于快速咨询

### 英雄部分
- 全屏背景展示区
- 主标题和副标题
- 突出的CTA按钮

### 服务分类
展示三个主要服务：
1. **短期游学** - 2-4周集中项目
2. **私校插班** - 1-2学期学期交流
3. **暑期学校** - 4-8周暑期项目

### 咨询功能
- 多个咨询触发点
- 模态框咨询表单
- 电话咨询选项
- 表单验证和提交处理

## 客户化建议

### 添加真实图片
1. 将英国/牛津/剑桥的图片放入 `images/` 文件夹
2. 更新 `css/styles.css` 中的 `.hero-background` 背景图片URL

### 修改联系信息
编辑 `index.html` 中的:
- 电话号码
- 邮箱地址
- 公司地址

### 配置后端API
更新 `js/script.js` 中的表单提交处理，连接实际的后端API
